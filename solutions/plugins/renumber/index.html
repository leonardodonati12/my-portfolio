AutoRenumber Pro

AutoRenumber Pro was developed to solve one of the most tedious tasks in BIM documentation: renumbering elements based on a specific path. Instead of manually clicking hundreds of instances, this plugin allows users to draw a spline through the elements, and the algorithm automatically detects intersections and renumbers them sequentially.

Built entirely with the Revit API and C#, this tool drastically reduces human error and saves hours of manual labor. It supports custom prefixes, suffixes, and padding, giving full control over the parameter output directly from an intuitive WPF interface.
